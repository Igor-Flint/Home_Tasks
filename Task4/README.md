# README: Основные команды Unix/Linux

## Команда `ls`

**Назначение:** Отображение содержимого директорий

**Синтаксис:**
```bash
ls [опции] [файлы/директории]
```

**Основные опции:**
- `-l` - детальный вывод (права, владелец, размер, время)
- `-R` - рекурсивный обход поддиректорий
- `-g` - показать имя группы владельца

**Примеры:**
```bash
ls                    # Содержимое текущей директории
ls -l                 # Детальный список
ls -la ~/Documents    # Все файлы в Documents с деталями
ls -R /var/log        # Рекурсивный обход
ls *.txt              # Только txt файлы
```

## Команда `echo`

**Назначение:** Вывод текста в терминал

**Синтаксис:**
```bash
echo [опции] [строка]
```

**Основные опции:**
- `-n` - не выводить перевод строки
- `-e` - интерпретация escape-последовательностей

**Escape-последовательности:**
- `\n` - новая строка
- `\t` - табуляция
- `\\` - обратный слеш

**Примеры:**
```bash
echo "Hello World"           # Простой вывод
echo -n "No new line"        # Без перевода строки
echo -e "Line1\nLine2"       # Две строки
echo $HOME                   # Вывод переменной окружения
echo *                       # Вывод всех файлов в директории
```

## Команда `cmp`

**Назначение:** Сравнение двух файлов побайтово

**Синтаксис:**
```bash
cmp [опции] файл1 файл2
```

**Коды возврата:**
- `0` - файлы идентичны
- `1` - файлы различаются
- `2` - ошибка

**Примеры:**
```bash
cmp file1.txt file2.txt      # Простое сравнение
```

## Команда `cp`

**Назначение:** Копирование файлов и директорий

**Синтаксис:**
```bash
cp [опции] источник назначение
cp [опции] источник... директория
```

**Примеры:**
```bash
cp file1.txt file2.txt      # Копирование файла
```

---

## Команда `cat`

**Назначение:** Объединение и вывод файлов

**Синтаксис:**
```bash
cat [опции] [файлы...]
```

**Основные опции:**
- `-n` - нумеровать строки

**Примеры:**
```bash
cat file.txt               # Вывод файла
cat file1.txt file2.txt    # Объединение файлов
cat -n script.sh           # Вывод с номерами строк
```

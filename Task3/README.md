## Описание программ

##Task3main.c - головной модуль

- Принимает в stdin строки

- Разбивает каждую строку на слова и "специальные символы" ( `|`, `||`, `&`, `&&`, `;` , `>`, `>>`, `<`, `(`, `)`)

- Показывает:
	- количество слов
	- полученные слова в лексикографическом порядке


## Управление

- **Ввод**: вводите строки для обработки (строка заканчивается переходом на новую строку `\n`)
- **Вывод**: `Ctrl+D`


## Описание функций

### Функции парсинга (`Task3main.c`)

#### `void nullbuf()`
**Назначение**: подготовить чистый буфер для нового слова
**Действие**:
- освобождает старую память, если буфер уже существовал
- выделяет новый буфер размером SIZE байт
- устанавливает текущий размер буфера
- сбрасывает указатель позиции в начало буфера
- инициализирует пустую строку

#### `void addsym()`
**Назначение**: безопасное добавление символа с автоматическим увеличением буфера
**Действие**:
- увеличивает размер буфера на константу SIZE, если буфер переполнен
- перевыделяет память с новым размером
- обновляет указатель на новый буфер
- добавляет символ в буфер и увеличивает позицию

#### `int symset(int c)`
**Назначение**: определить, можно ли добавить символ к текущему слову
**Параметы**:
- 'c' - числовая переменная
**Действие**:
- Возвращает 1 (true), если символ НЕ является разделителем (`\n`, ` `, `\t`, `EOF`) и не является специльным повторным символом (`&`, `|`, `>`)

#### `int getsym()`
**Назначение**: получение следующего символа
**Действие**:
- Проверка необходимости чтения нового блока
- Чтение нового блока данных
- Обработка символа после блока
- Обработка пустого чтения
- Возврат символа

#### `void finish_word()`
**Назначение**: завершение текущего слова
**Действие**:
- проверка необходимости увеличения буфера для '\0'
- добавляет слово в список, если оно не пустое

#### `void* start()`
**Назначение**: НАЧАЛЬНОЕ СОСТОЯНИЕ
**Действие**:
- в зависимости от очередного символа выполняет определенные функции

#### `void* word()`
**Назначениnewline()е**: состояние чтение простого слова
**Действие**:
- если символ не специальный, то остаемся в состоянии `word()`. В противном случае завершает слова и переходит в `start()`

#### `void* arrow()`, `void* amper()`, `void* vertStick()`
**Назначение**: определить, специальное слова "двойное" или "одинарное"
**Действие**:
- Проверяет следующий символ на возможность образования двойного оператора

#### `void* newline()`
**Назначение**: обработка новой строки
**Действие**:
- завершает текущее слово
- освобождает старый список и создает новый
- возвращается `start()`

#### `void* stop()`
**Действие**: завершает работу программу


### Функции работы со списком (`Task3struct.c`)

#### `void create_list(list_of_lines *list)`
**Назначение**: инициализация списка
**Параметры**:
- `list` - указатель на структуру list_of_lines
**Действие**:
- устанавливаются начальные значения в `length` и в `capacity`
- выделяет память и заполняет все ячейки NULL

#### `void add_to_list(list_of_lines *list, const char *str)`
**Назначение**: добавление нового слова в список
**Параметры**:
- `list` - указатель на структуру list_of_lines
- `str` - указатель на строку с очередным словом
**Действие**:
- Проверка и увеличение емкости при необходимости
- Выделение памяти для новой строки
- Копирование строки и обновление длины

#### `void termlist(list_of_lines *list)`
**Назначение**: корректное завершение списка
**Параметры**:
- `list` - указатель на структуру list_of_lines
**Действие**:
- Увеличение емкости при необходимости
- Установка NULL-терминатора

#### `void sort_list(list_of_lines *list)`
**Назначение**: сортировка (пузырьком)
**Параметры**:
- `list` - указатель на структуру list_of_lines

#### `void free_list(list_of_lines *list)`
**Назначение**: освобождение памяти
**Параметры**:
- `list` - указатель на структуру list_of_lines
**Действие**:
- Освобождение отдельных строк
- Освобождение массива указателей
- Сброс состояния структуры

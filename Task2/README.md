##Описание программ

##1.c - Метод Ньютона

Вычисляет квадратный корень числа с заданной точностью.

Ввод: точность e, затем последовательность чисел
Вывод: значения квадратных корней

##2.1.c - Схема Горнера

Вычисляет значение полинома в заданной точке.

Ввод: значение x, затем коэффициенты полинома
Вывод: значение полинома в точке x

##2.2.c - Полином и производная

Вычисляет значение полинома и его производной в точке.

Ввод: значение x, затем коэффициенты полинома
Вывод: значение полинома и производной

##3.c - Числа Фибоначчи

Вычисляет n-ое число Фибоначчи итеративным и рекурсивным методами.

Ввод: последовательность целых чисел n
Вывод: числа Фибоначчи F(n)

##4.c - Преобразование строки в double

Реализует функцию преобразования строки в число типа double.

Ввод: строки с числовыми значениями
Вывод: преобразованные числа

##5.c - Обработка списка

Работа с односвязным списком слов.

Программа читает входную строку, разбивает её на слова и сохраняет в односвязный список. Затем удаляет из списка все слова, совпадающие с последним словом (кроме самого последнего слова).
Ввод: строка, содержащая слова, разделённые пробелами.

Вывод: список слов после удаления всех вхождений последнего слова (кроме последнего).

##6.c - Дерево поиска

Реализация двоичного дерева поиска с операциями добавления, удаления и проверки наличия элементов.

Ввод: команды формата +число, -число, ?число
Вывод: для команды ? выводит yes или no в зависимости от наличия числа в дереве

Операции:

+число - добавить число в дерево
-число - удалить число из дерева
?число - проверить наличие числа в дереве

##7.c - Калькулятор выражений

Расширенный калькулятор арифметических выражений с поддержкой операций разного приоритета.

Ввод: арифметические выражения
Вывод: результат вычисления
Поддерживаемые операции:

+ - сложение (левый приоритет)
- - вычитание (левый приоритет)
* - умножение (левый приоритет)
/ - целочисленное деление (левый приоритет)
^ - возведение в степень (правый приоритет, высший приоритет)
( ) - скобки для изменения приоритета
Грамматика:

<выражение>   ::= <слагаемое> { (+|-) <слагаемое> }
<слагаемое>   ::= <множитель> { (*|/) <множитель> }
<множитель>   ::= <степень> { ^ <степень> }
<степень>     ::= цифра | ( <выражение> )
